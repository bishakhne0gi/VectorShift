# VectorShift

Usage of advance design-pattern: Compounded Components Design Pattern


## Installation

Frontend

```bash
  cd frontend
  npm i
  npm start
```

Backend

```bash
  cd backend
  uvicorn main:app --reload 
```  



## Tech Stack

**Client:** React, ReactQuey, Zustand, Ky,  TailwindCSS

**Server:** FastApi
